# System Bank Backend

## Opis Projektu

**System Bank Backend** to serwer stworzony przy użyciu Django i Django Rest Framework, zapewniający RESTful API do obsługi zarówno Systemu Bankowego, jak i Systemu Bank Admin GUI. Projekt umożliwia komunikację pomiędzy frontendem (System Bank Admin GUI) a backendem (System Bank Backend) poprzez API, co umożliwia obsługę funkcji bankowych.

## Funkcje

1. **Django Rest Framework API:**
   - Projekt wykorzystuje Django Rest Framework do stworzenia pełnowartościowego interfejsu API obsługującego System Bank oraz System Bank Admin GUI.

2. **Obsługa Kont Bankowych:**
   - Tworzenie, odczytywanie, aktualizowanie i usuwanie (CRUD) kont bankowych.

3. **Zarządzanie Użytkownikami:**
   - Rejestracja nowych użytkowników.
   - Uwierzytelnianie i autoryzacja użytkowników.

4. **Obsługa Wniosków:**
   - Tworzenie i przeglądanie wniosków o utworzenie konta.

5. **Statystyki Systemu:**
   - Udostępnianie statystyk dotyczących działania Systemu Bankowego.

6. **Integracja z Systemem Bank Admin GUI:**
   - Zapewnienie interfejsu dla Systemu Bank Admin GUI do komunikacji z backendem.
